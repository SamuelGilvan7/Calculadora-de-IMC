# Calculadora de IMC

Projeto desenvolvido para praticar conceitos fundamentais de JavaScript, manipulação do DOM, eventos, funções, validações e interação com formulários HTML.

## 📌 Sobre o Projeto

A Calculadora de IMC (Índice de Massa Corporal) permite que o usuário informe seu peso e altura para obter automaticamente o valor do IMC e sua respectiva classificação.

O projeto foi desenvolvido com foco em estudos e prática de desenvolvimento web utilizando HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

* Cálculo automático do IMC.
* Validação dos campos de peso e altura.
* Exibição de mensagens de erro para valores inválidos.
* Exibição da classificação do IMC.
* Interface simples e intuitiva.
* Tabela de referência das classificações do IMC.

## 📊 Classificações do IMC

| IMC            | Classificação    |
| -------------- | ---------------- |
| Menor que 18.5 | Abaixo do peso   |
| 18.5 até 24.9  | Peso normal      |
| 25 até 29.9    | Sobrepeso        |
| 30 até 34.9    | Obesidade grau 1 |
| 35 até 39.9    | Obesidade grau 2 |
| 40 ou mais     | Obesidade grau 3 |
